# Tic Tac Toe

## Description
This is a simple **Tic Tac Toe** game where the player is **X** and the computer is **O**. The game checks for a winner after every move and handles draw conditions.

---

## Features
- **Player vs Computer** gameplay
- **Winner detection**
- **Draw detection**
- **Reset button** to restart the game

---

## How It Works
- When the player clicks a box, **X** is placed.
- The computer selects a random empty box and places **O**.
- The game checks predefined winning patterns after each move.
- If all boxes are filled and no winner is found, the game is a **draw**.
